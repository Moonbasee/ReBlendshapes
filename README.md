# ReBlendshapes

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) that automatically creates ValueCopies between your avatar's blendshapes and it's clothing/other accessory.

## Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
1. Place [ReBlendshapes.dll](https://github.com/Moonbasee/ReBlendshapes/releases/latest/download/ReBlendshapes.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create this folder for you.
1. Start the game. If you want to verify that the mod is working you can check your Resonite logs.

## How To Use
1. Go To Your Avatar's Body Slot
1. Scroll Down To The End Of The SkinnedMeshRenderer And Place The SkinnedMeshRenderer Of The Accessory In The 'Skinned Mesh Renderer' Field
1. Click 'Create Value Copies'
1. Profit
